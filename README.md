#Histogram

An html version of a [Histogram](http://en.wikipedia.org/wiki/Histogram).


##Install
> bower install histogram



##Usage

> histogram( id, options )

See index.htm for example usage, but this,

    <div id="h1"></div>
    <script>
        histogram( "h1", {
                   "data": [70,20,30,40,10]
                  } );
    </script>

will get you going.
